# hot-or-not
[Tinder](https://tinder.com/) like SPA mobile web-app

## Stack
 - Vue
 - Vuex
 - Vue-Router
 - SCSS
 - Eslint

## Code style
 - @vue/airbnb


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
